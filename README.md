# Take Water - Admin Panel

Interactive admin panel prototype for the **Take Water** purified water delivery app.

## Features

- 🎨 **Take Water Blue Theme** - Matches mobile app branding
- 📊 **Dashboard** - Sales charts, stats cards, top products
- 📦 **Product Management** - CRUD with search/filter
- 👤 **Customer Management** - Customer database with balances
- 🛒 **Order Management** - Order tracking with status
- 📊 **Stock Management** - Inventory across shops
- 🚚 **Supplier Management** - Supplier database
- 👥 **Staff Management** - Staff with roles
- 📂 **Category Management** - Product categories
- 📈 **Reports** - Multiple report types
- ⚙️ **Settings** - Company & shop configuration
- 🔐 **Login Page** - Authentication UI
- 📱 **Responsive** - Works on desktop & tablet

## Tech Stack

- HTML5
- CSS3 (CSS Variables for theming)
- Vanilla JavaScript
- Chart.js (CDN)

## How to Use

1. Open `index.html` in your browser
2. Click "Login" to access the admin panel
3. Navigate using the sidebar menu
4. Click "Add" buttons to see modal forms
5. All data is mock data for demonstration
